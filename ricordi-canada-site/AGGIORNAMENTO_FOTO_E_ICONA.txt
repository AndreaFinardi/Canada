AGGIORNAMENTO: ALBUM FOTO + ICONA DEL SITO
==========================================

Questa versione aggiunge:
- sezione "Le nostre foto" divisa per persona;
- album iniziali Ilaria e Sofia;
- visualizzazione grande delle foto con frecce avanti/indietro;
- icona del sito nella scheda del browser e sulla Home del telefono;
- supporto per creare tutti i gruppi che vuoi.

COME AGGIORNARE GITHUB
----------------------
1. Estrai lo ZIP.
2. Apri su GitHub la cartella ricordi-canada-site.
3. Carica i file contenuti nella nuova cartella ricordi-canada-site,
   mantenendo esattamente le stesse sottocartelle e sostituendo i file esistenti.
4. Conferma il commit.
5. Render avvierà automaticamente un nuovo deploy.

Non devi modificare Root Directory, Build Command o Start Command.

DOVE METTERE LE FOTO
--------------------
Foto di Ilaria:
private/photos/ilaria/

Foto di Sofia:
private/photos/sofia/

Poi modifica:
data/photos.json

Esempio di una foto:
{
  "file": "ilaria-03.jpg",
  "alt": "Foto con Ilaria",
  "caption": "Una giornata speciale"
}

Per aggiungere una nuova persona, crea per esempio:
private/photos/martina/

e duplica un gruppo dentro data/photos.json cambiando id, nome e fotografie.
